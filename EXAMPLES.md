# Examples — OpenTUI (Workspace Root)

This `examples` folder holds runnable example apps for OpenTUI.

- React examples: `examples/react/*` — See `examples/react/README.md` for info.

How to run an example (from repo root):

```bash
bun install
bun examples/react/basic.tsx
```

If you need more detailed documentation, check `opentui/packages/react/docs/EXAMPLES.md`.
